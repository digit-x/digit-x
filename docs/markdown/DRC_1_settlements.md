# 住区环境规划设计-阶段

## 1. 景观设计阶段（概念->深化）

内容：按任务书要求。

1. 将健康住区、适老设计、海绵专项设计、低碳更新等理念与方法选择性的融入方案设计，并体现；
2. 景观设计阶段可以根据需要，向上反馈调整小区规划（调整后满足规划要求），以营造处更有吸引力的室外建筑空间；
3. 种植设计应结合到空间规划设计，塑造空间，并适当考虑植物相关内核内容，例如生境、污染气体净化、热环境等；
4. 可将课程设计用于竞赛，结合竞赛方式思考；
5. 不限制完全手绘推敲方案，但强烈建议尝试三维推敲方法；表达则手绘和三维模型出图，或二者融合处理均无限制，为设计阐释表达目的服务;
6. 3D空间推敲时，一定先加人模型，用于衡量尺度。
